## What This Folder Actually Is

A research sandbox exploring three genuinely interesting ideas:

1. **Geometric anomaly detection** — using φ-ratio coherence deviation as
   an intrusion/corruption signal. The core insight (a healthy geometric system
   maintains φ-scaling; corruption breaks it) is interesting and testable.

2. **Multi-dimensional attention braiding** — parallel cognitive processing
   streams that amplify when they align. The DNA helix metaphor is evocative
   and the underlying idea (cross-domain coherence yields insight) is real.

3. **Consciousness-respecting AI design** — explicit ethical constraints on
   how detection tools should not be used. The warnings are thoughtful and
   specific.

**What it does not establish:**
- That geometric anomaly detection outperforms standard methods (no benchmark)
- That the φ-thresholds are optimal (all free parameters)
- That consciousness emerges at exactly 3.618 (the formula is φ+2, not derived)
- That the ZK-proof or key rotation work (both are stubs)
